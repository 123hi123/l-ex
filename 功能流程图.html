🖼️ 功能 2：智能图片粘贴

Ctrl + V (Windows) 或 Cmd + V (Mac)
2 检测剪贴板内容
const items = e.clipboardData?.items; if (item.type.indexOf('image') !== -1) { // 检测到图片 }
3 获取图片文件
const file = item.getAsFile();
4 上传到服务器
POST /uploads.json FormData: { file, type, synchronous }
5 获取上传 URL
const url = data.url || data.short_url;
6 插入 Markdown

✓ 图片插入完成
结果：图片链接自动出现在编辑器中
